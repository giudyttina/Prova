Descrizione

Object type: Autel votif ou base de statue
Materiale: calcaire
Tipologia: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Probablement du IIe siècle, en raison des belles lettres(Hirschfeld).

Text

I(ovi) o(ptimo) m(aximo)
P(ublius) Aelius
Mucia
nus
5v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon(et) très grand. Publius Aelius Mucianus s’est acquitté de(son) vœu de bon gré(et) à bon droit.


Commentary:

Le cognomen Mucianus ne dérive pas du nomen Mucius, mais bien du nom thrace Muca ; voir Kerényi, Russu.
